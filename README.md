# react-native-iot-wifi

Connect to WiFi with React Native on Android and iOS.

## Installation

```sh
npm install react-native-iot-wifi
```

## Usage

```js
import { multiply } from 'react-native-iot-wifi';

// ...

const result = await multiply(3, 7);
```
